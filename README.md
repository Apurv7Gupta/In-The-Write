# In The Write

**A blogging website** that allows you to write, share, and save blogs directly to your device.

## Features
- 🌙 **Dark Mode**
- 🖼️ **Randomly Updated Blog Gallery**

## Tech Stack
- ⚛️ **React**
- 🎨 **TailwindCSS**
- 🔄 **ContextAPI**
- 📸 **Lorem PicsumAPI**
